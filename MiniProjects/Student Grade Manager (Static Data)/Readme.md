# Student Grade Manager

C# program that shows student names, grades, and calculates average.

## What I Used

- Arrays
- For loop
- Static methods
- LINQ Average()

## Run

```bash
dotnet run
```

## Output

```
Student Grades:
================
Name: Ali, Grade: 90
Name: Sara, Grade: 55
Name: Omar, Grade: 40

Average: 61.666668
```
